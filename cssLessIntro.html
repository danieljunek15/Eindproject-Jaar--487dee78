CSS

Met behulp van een cascading style sheet (CSS) kan je opmaak toevoegen aan je website. Dit gebeurd door per element te beschrijven hoe deze eruit moet komen te zien. Zoals je hebt gezien heeft je webbrowser standaard al een beetje styling in zich, bijvoorbeeld dat de koppen (h1, h2 etc.) dikgedrukt moeten zijn. Met de eigen CSS kan je dit overschrijven.
